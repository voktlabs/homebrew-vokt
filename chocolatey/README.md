# Chocolatey Package for Vokt

This directory contains the Chocolatey package configuration for Vokt.

## Installation

### From GitHub Releases

Download the latest `.nupkg` file from [releases](https://github.com/voktlabs/homebrew-vokt/releases) and install:

```powershell
# Navigate to the directory containing the .nupkg file
choco install vokt-cli -s . -y
```

## Verify Installation

```powershell
vokt --version
```

## Upgrade

```powershell
# Download the new version .nupkg file and run:
choco upgrade vokt-cli -s . -y
```

## Uninstall

```powershell
choco uninstall vokt-cli -y
```

## Package Maintenance

The package is automatically generated by the CI/CD pipeline when a new release is created in the [vokt repository](https://github.com/voktlabs/vokt).

### Package Structure

- `vokt.nuspec` - Package metadata
- `tools/chocolateyinstall.ps1` - Installation script (downloads binary, Java engine, Go toolchain)
- `tools/chocolateyuninstall.ps1` - Uninstallation script
